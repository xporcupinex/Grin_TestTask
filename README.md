# Hero Screen UI - прототип адаптивного окна персонажа для мобильной игры

Сцена с Canvas находится здесь Assets/Scenes/TestTaskScene.unity

Сейчас Canvas Scaler адаптирован под горизонтальную ориентацию 16:9 - Match Height 1, для адаптации под планшеты 4:3 я бы предложила попросить написать программиста скрипт, который при старте игры проверяет соотношение сторон экрана и сторон ставит разные настройки Canvas Scaler в зависимости от того вытянутый ли это мобильный ли это телефон или ближе к квадрату планшет.

Адаптивная верстка проверялась на разрешении макета 2024x1080, стандартном FullHD 1920x1080, и более вытянутом 2400x1080
